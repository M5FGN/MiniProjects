# Magic 8 Ball

## Description

Digital recreation of the classic Magic 8 Ball. 

## MVP

* Create a Magic 8 Ball visual.
* Display random answers from the list of standard answers.
* Be able to reset the Magic 8 Ball.

## Technologies Used

* HTML
* CSS
* JavaScript
